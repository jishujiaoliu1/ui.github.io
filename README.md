How to learn network programming?
==
Suppose I have trained an object detection model in Python, and I also use a Python program (under Windows environment) when testing. Now I want to move this set of things to the server. When the Android App uploads a picture to the server, the server can process the picture and return the label of the picture, such as "cow", back to the Android App.

question:

① What system should I choose for the server? (I have no Linux programming experience)

② What is the rough implementation plan?

③ What relevant knowledge and technologies do I need to learn in order to implement this plan?

④ If I adopt a learning method that emphasizes process over principles, can I learn and achieve the results I want within two months? I want to take advantage of the holiday to learn a small project by myself and experience it. I will be busy after the new year and won’t have much time to study by myself.

My technical background: I am a second-year undergraduate student majoring in computer science. I have learned some C/C++, Java, Python, data structures and probability statistics, but I have not learned computer networks yet.
